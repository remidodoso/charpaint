# charpaint

A MacPaint-style character-cell painting app, built in Rust and compiled to WebAssembly.

This is an experiment for me. I wrote a version of Charpaint long ago on Mac in the late 90s. This, though, is coded using Claude prompts in VS Code. I don't know much Rust and I know almost nothing about WASM, so I thought, let's see if I can "code" a new version anyway.
